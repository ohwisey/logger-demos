The base

This first prompt sets the whole look. Every prompt after inherits it, so your logger comes out on-theme without repeating yourself.

Why this prompt is longer

It carries the theme passport: the exact colors, font, and feel. Set it once here and every later prompt stays two lines. This is the trick that makes the whole build look designed instead of default.

0The single-file shelltheme passport · offline · saves in the browser 
Build a workout app as ONE single index.html file. No build step, works offline,
saves everything in the browser so it is still there when I reload.

THE THEME. Follow this exactly, for everything we add from now on:
- Background pure black (#000). No grays, no gradients on the page itself.
- One accent: mint (#6EE7B7). Mint is only for wins, live numbers, and the
  primary action. Never decorate with it.
- Font: Inter for everything. Numbers bold and big, labels small and quiet.
- Big rounded cards (18px), hairline borders (white at 8%), lots of breathing room.
- Mobile first. Big tap targets, everything reachable with a thumb.
- The overall feel: a premium Apple Health app. Calm, spacious, one thing at a time.

For now just build the shell: a clean title bar and an empty "today" area below it.
Nothing else yet. We add one piece at a time.
Part 1 · progressive overload

The proof you are winning

1.1Log a liftweight · reps · saved history 
Let me pick an exercise and log sets with weight and reps.
Keep the full history of every set per exercise. Same theme.
1.2The session to beatyour last best, always visible 
Next to each exercise show my best previous set. The moment a new set beats it,
light that set up in mint. Winning should be visible instantly.
1.3The chartthe line that climbs 
Add a clean mint line chart of my top set over time for each exercise.
Let me scrub across it with my finger to see any past session.
Under it: best ever, and my current beat-last-time streak.
Part 2 · the workout logger

Grades every set, live

2.1Today's sessionthe full workout screen 
Build a "today" screen: I add exercises to a session and log several sets under
each, weight and reps. Each exercise is its own card. Same theme.
2.2Grade every setinstant feedback on save 
The instant I log a set, grade it against my last time on that exercise:
a small badge on the set row. Beat it = mint. A personal record = amber "PR".
Below it = quiet gray, never red, never shame. Make it feel alive but calm.
2.3Finish the daysave it · a small earned celebration 
Add a Finish button that saves the session and shows a short celebration screen:
what I lifted today and anything I beat. Understated, earned, not confetti spam.
Part 3 · the exercise library

Teaches as you build

3.1The librarysearchable · by muscle group 
Add a searchable exercise library I can open while building my session.
Organize it by muscle group with a search box on top. Include the classics
(bench, squat, deadlift, rows, presses, curls) so it is useful from minute one.
3.2Teach the movehow-to + muscle diagram 
When I tap an exercise in the library, show a simple how-to (3 short steps)
and a small diagram of the muscle it hits, so I learn the move as I pick it.
3.3Add and swapinto today, without losing history 
Let me add an exercise from the library straight into today's session, and swap
one exercise for another without losing my logged sets or my history.
