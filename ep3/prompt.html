01 build-the-library.txt PROMPT

Build me a single-file exercise library I can open in my browser. One page, and make it look good: pure black background, white and soft grey text, with one mint green as the only accent color. Use a classy serif for the titles, something like Instrument Serif, and a clean sans like Inter for everything else. Rounded cards, lots of breathing room, calm and premium, no other bright colors. Show every move grouped by the muscle it trains, with a search box at the top. I tap a move to add it, tap again to remove it. If I search for something that is not there, let me add my own by name from the same box. Each move has its own card: a picture, three short steps to do it right, and a couple of cues that matter. Let me swap one move for another. Opening the swap opens this same library, and when I pick the new move it takes the old one's place and keeps my history, no duplicate. Give me a kg and lb toggle in the corner: store one unit, convert at the edge, and let plus and minus step by a clean plate while the middle stays a number I can type. Save everything in the browser so it is still there when I come back. Seed it with a few real moves so it feels alive on first open. Make it feel like an app I would pay for.

02 back-up-to-supabase.txt PROMPT

Take the app you just built and back it up to Supabase so my data follows me to any device, not just this one browser. Walk me through it step by step. If I do not have a Supabase account yet, help me make a free one, click by click. Create only the one table this app needs and turn on row level security so only I can see my own data. Show me exactly where to paste my project URL and key. Then wire the app to read and write from Supabase, and keep it working offline on my device if I ever lose connection. Keep it to one table and the fewest steps.
